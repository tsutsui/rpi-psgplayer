## Raspberry Pi 3 / Zero YM2149F PSG Player 回路図 補足

- Rapsberry Pi の GPIO は 3.3V、 YM2149F 側は 5V ですが、
  ドライバ仕様として Rapsberry Pi → YM2149F への書き込みのみで
  YM2149F → Raspberry Pi 方向の読み出しはしないこと、
  YM2149F の各端子の H入力しきい値はすべて 2.2V 以上であることから
  「とりあえず動くし、たぶん壊れはしない」という設計です。
  実装を誤って YM2149F を出力にしてしまった場合など
  **最悪時は Raspberry Pi 側の GPIO 周辺が破損する可能性もありますが
  そこはお約束の自己責任ということでお願いします**

- [RPi-PSG_v2_schematic.pdf](RPi-PSG_v2_schematic.pdf) は
  OSC大阪で展示デモした大型基板の Raspbery Pi 3B バージョンですが
  HAT版のクロック追加に合わせて GPIO のピン割り当ては更新しています

- [RPi-PSG-HAT_schematic.pdf](RPi-PSG-HAT_schematic.pdf) は
  OSC東京で Rapsberry Pi Zero に接続して展示していた HATサイズ版で、
  PSGチップ用 2MHz クロックを Raspberry Pi GPCLK0 から供給する版です

- YM2149F と Rapsberry Pi GPIO 間の接続は
  [psg_backend_rpi_gpio.c](../psg_backend_rpi_gpio.c) の先頭コメントを
  見たほうがわかりやすいと思います

- YMZ284, YMZ294 等を使った同等の回路説明では
  一部の端子 (リセットなど)の Hしきい値が 3.3Vより上にある、
  という記載があると思いますが、 YM2149F (および AY-3-8910) では
  そのような制約というか仕様のピンはありません

- が、RESETピンについては H側が通常時、リセット時に L駆動なので、
  3.3V のままで保持するにはちょっと気持ち悪かったので
  内部プルアップ前提のトランジスタオープンコレクタ駆動
  (論理としては反転) にしてあります
  (Lしきい値は 0.8V なのでそこまで気にする必要はないのですが気分の問題)

- 3B 版について OSC東京展示デモ向けに DA0〜DA7 に LED表示を追加しましたが
  回路図には記載していません (DAx - 330Ω - GND とつないでいるだけ)

- 3B 版のジャンパは 4MHz or 2MHz クロック切り替え用です。
  詳細は YM2149F のマニュアルを参照してください。
  2MHz クロック使用時は AY-3-8910 も使用可能です。
  (最初 4MHz 発振器しか手持ちになかったので YM2149F を使った、という順番)

- YM2149F 右の出力側は Raspberry Pi とは関係ない 3ch 出力加算+アンプですが
  3B バージョンはオペアンプ加算+LM386アンプというほぼ教科書通りの回路です

- 3B版の半固定抵抗とステレオジャックは確か千石電商で買ったもののはずですが
  店舗で現物を見て買ったものなので型名詳細について情報が残っていません

- HAT版の出力回路は実装面積削減のため抵抗オンリーのパッシブ合成と
  [秋月電子製 PAM8012 D級アンプ](https://akizukidenshi.com/catalog/g/g108217/)
  という構成です

- HAT版の RV1 の半固定抵抗は秋月電子で販売されている
  [SUNTAN TSR-3386T-EY5-503TR](https://akizukidenshi.com/catalog/g/g108013/)
  を使っていますが、
  [商品ページの画像](https://akizukidenshi.com/img/goods/2/108013.jpg)
  にある通り、通常の可変抵抗とは異なり
  「ボリュームを物理的な1番端子の方向に回すと電気的な接点は3番端子側に近づく」
  という構造なので配線時は要注意です

- HAT版の J2 ステレオジャックは品番としては秋月電子で売っている
  [マル信無線電機製 MJ-352W-O](https://akizukidenshi.com/catalog/g/g108958/)
  ですが、接続するスピーカーに合わせて直結含め都合のいいものを使ってください

- HAT版の JP1 のジャンパはステレオジャックにモノラルプラグを差す場合はオープン、
  ステレオプラグを差して左右両方から音を出したい場合はショートさせてください。
  ただし、ステレオスピーカー接続時は8Ω以上（＝並列接続で4Ω以上）のものでないと
  たいていのアンプは駆動しきれないと思います
